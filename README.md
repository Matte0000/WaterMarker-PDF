# WaterMarker-PDF
WaterMarker of PDFs file.

# How it works
Perform these steps:
```
Fork this repo
Clone this repo
cd watermarker-pdf
```
On terminal run this command:
```
python watermarkerpdf.py <watermark_file> <input_file> <output_file>
```

3. How to Read, Write, Merge PDF file
